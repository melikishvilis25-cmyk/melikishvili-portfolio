# David Melikishvili - Portfolio

My personal portfolio website built from scratch using HTML, CSS, and JavaScript.

This project is a way for me to showcase my skills, projects, and progress as I continue learning software development.

## 🌐 Live Website

https://melikishvili.netlify.app/

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## ✨ Features

- Responsive portfolio layout
- Animated tab switching system
- Project showcase section
- Social links
- Contact email link
- Custom styling and animations
